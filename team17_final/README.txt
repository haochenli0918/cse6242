GTRent is a web application which offers better rental options for Gatech students. 

The CODE folder contains all the code of the application. Under the CODE folder, the src folder contains a sub-folder called data which holds our final data, all the javascript files of the web application and the python files we used to analyze data. The public folder contains the nessasary pictures and icons. 
The DOC folder contains the final report and the poster where details of our project could be found.

Enter CODE folder and run npm install to set up our application. Run npm start to run it on localhost.
